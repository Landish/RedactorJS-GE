#RedactorJS-GE

Redactor WYSIWYG HTML Editor Georgian Language Pack


#Usage

Simply include `ge.js` file on your page after loading `redactor.js` and set `lang` option to `ge` in your Redaqtor settings.

```
<script src="js/ge.js"></script>
<script>
$(function() {
  $('#redactor').redactor({
    lang: 'ge'
  });
};
</script>
```
